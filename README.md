# FocusGuard: 第一性原理防分心复盘守护程序 👁️



> **“物理外挂”级别的专注力守护工具。**
> 当你坐在电脑前“惯性摸鱼”或在“负面心流”中偏离核心目标时，FocusGuard 会毫无预兆地跳出来，用当时的屏幕截图“审判”你，并强制你进行第一性原理复盘。

---

## 🎯 为什么需要 FocusGuard？

在工作或学习中，我们的大脑极其擅长自我欺骗：
- 虽然在刷网页，但心里觉得自己是在“查资料”。
- 虽然在做杂事，但心里觉得自己“很忙”。
- 固定时间的闹钟很容易产生肌肉记忆，被随手点掉。

**FocusGuard 的逻辑是：**
你永远不知道下一次“拷问”什么时候到来。它会在后台静默观察，在随机的时间点突然锁定你的屏幕，并拍下你正在干活的证据。你必须面对截图，诚实地回答：**我现在的行为在服务于我的根本目标吗？**

---

## ✨ 核心特性

- **🎲 薛定谔定时器**：在 30~90 分钟（可配置）的区间内**完全随机**触发，彻底打破肌肉记忆和心理预判。
- **📸 截屏取证机制**：弹窗触发前一秒静默全屏截图。截图直接展示在复盘窗口，防止任何形式的抵赖和自我欺骗。
- **🔒 物理级焦点劫持**：弹窗采用置顶全屏暗幕，屏蔽 `Esc`、`Alt+F4` 等常规关闭手段。你必须输入至少 3 个字的有效描述才能重获自由。
- **📔 离线处刑日志**：所有复盘记录和截图按日期归档在本地 `data/` 目录。支持 Markdown 格式，每日睡前回顾，偏航轨迹一目了然。
- **👁️ 系统托盘驻留**：紫色“守护之眼”图标常驻托盘，支持休眠模式（开会专用）、查看今日记录等功能。
- **📦 极简分发**：支持一键构建 **18MB** 的极致瘦身版 `.exe`，解压即用，无需 Python 环境。

## 🚀 快速开始

### 方式一：直接使用（推荐）
如果你不熟悉编程，直接前往 [Releases](https://github.com/Lin0371/FocusGuard_compact) 下载 `FocusGuard_compact.exe`，双击启动即可。

### 方式二：开发者模式
1. **克隆项目**
   ```bash
   git clone https://github.com/Lin0371/FocusGuard.git
   cd FocusGuard
   ```
2. **安装依赖**
   ```bash
   pip install -r requirements.txt
   ```
3. **启动程序**
   ```bash
   # 后台静默启动（自动脱离控制台）
   python main.py
   ```

## ⚙️ 个性化配置

修改 `config.py` 来适配你的节奏：

```python
MIN_INTERVAL = 30 * 60   # 最短随机间隔（秒）
MAX_INTERVAL = 90 * 60   # 最长随机间隔（秒）
MIN_INPUT_LENGTH = 3      # 复盘描述最少字数要求
DEBUG = False              # 开启后间隔缩短至 10-20 秒，用于测试
```
